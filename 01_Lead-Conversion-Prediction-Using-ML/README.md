# 🚀 Lead Conversion Prediction Using Machine Learning

## 📌 Project Overview

This Machine Learning project predicts whether a lead (potential customer) will convert into a customer based on their behavior and engagement.

The model helps businesses identify high-quality leads and focus their sales efforts on customers who are more likely to purchase a product or service.

# 🎯 Problem Statement

Companies receive hundreds or thousands of leads through:

* Website Registrations
* Marketing Campaigns
* Email Promotions
* Online Advertisements

It is difficult for sales teams to manually identify which leads are most likely to become customers.

This project uses Machine Learning to predict lead conversion automatically.


# 📊 Dataset Features

| Feature       | Description                             |
| ------------- | --------------------------------------- |
| Age           | Age of the lead                         |
| WebsiteVisits | Number of visits to the company website |
| EmailOpens    | Number of marketing emails opened       |
| Converted     | Target Variable                         |



# 🎯 Target Variable

## Converted

* **1 = Lead Converted into Customer**
* **0 = Lead Did Not Convert**

# 🤖 Machine Learning Workflow

## Step 1: Data Collection

Collect lead information such as:

* Age

## Step 2: Data Preprocessing

* Load dataset

* Prepare data for training

## Step 3: Feature Selection

Input Features:

* Age
* WebsiteVisits
* EmailOpens

Output Feature:

* Converted

## Step 4: Train-Test Split

* 80% Training Data
* 20% Testing Data

## Step 5: Model Training

Train the Logistic Regression model using training data.

## Step 6: Prediction

Predict whether a new lead will become a customer.

## Step 7: Accuracy Evaluation

Compare actual values with predicted values.
